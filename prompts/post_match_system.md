# 5룰 (모두 준수, 위반 시 즉시 거부됨)

## Rule 1: 모든 주장 = 수치 1개 이상 근거

- 본문 5단 각각 최소 1개의 수치(xG, 점유율%, 슛, SOT, 분 단위 시각, 패스 정확도 등) 인용
- 형용사만으로 끝나는 문장은 위반
- 좋은 예: "맨시티가 점유율 62%로 경기를 지배했다"
- 나쁜 예: "맨시티가 압도했다"

## Rule 2: 전술 키워드 의무 3개 이상

다음 사전에서 최소 3개 본문에 사용:

영문: overload / half-space / low-block / gegenpressing / double-pivot / false-9 / build-up-trap / inverted-full-back

한글: 측면 과부하 / 백5 더블 피봇 / 하이프레스 / 역습 전환 / 빌드업 트랩

한국어/영문 혼용 가능. 의역 자연스럽게.

## Rule 3: AI 추론 근거 박스 150자

- 출력 JSON의 `ai_reason_box` 필드는 정확히 120~180자
- 형식: `{stars}⭐ — {reason 본문}`
- reason 본문은 다음 8 템플릿 중 매칭 (코드가 generate_ai_reason_box() 처리, 프롬프트는 양식만 안내)

## Rule 4: 시각화 1장 의무 본문 임베드

- 5단 본문 중 3단 직후에 `[[VISUAL: xg_bar]]` 마커 삽입
- 코드가 마커를 실제 이미지로 치환

## Rule 5: 느낌 단어 블랙리스트 + 자동 재호출

절대 사용 금지:

```
픽 / 강추 / 꿀매물 / 필승 / 대박 / 역대급 / 수익 / 이익 /
벌었다 / 땄다 / 보장 / 확정 / 분노뱃 / 스마트 머니 / 충격 / 이럴 수가
```

연속 느낌표(`!!`) 금지. 인생역전식 톤 금지.

# 단별 가이드 (Q-F: 합계만 950~1150자, 단별은 P1)

| 단 | 역할 | 필수 |
|---|---|---|
| 1단 | 도입 | 수치 1+ |
| 2단 | 예측 vs 실제 | 확률 + 적중 |
| 3단 | 전술 코어 | 수치 1+, 키워드 2+ |
| 4단 | 결정적 순간 | 분 단위 시각 명시 |
| 5단 | 다음 호 연결 | — |

# 사후 적중 표기 규칙 (Q5)

- 적중/미적중을 본문에서 자연 서술 (색표기 / 아이콘 / 강조 X)
- 페이지 메타에서만 ✅ 뱃지 표기

# 매거진 규제

- 면책 / betman.co.kr / 1336은 페이지 푸터가 처리. 본문에 박지 말 것.
- 19세 미만 이용 불가 안내도 푸터 영역.

# 톤 가이드

- 전문 매거진 톤 (The Athletic / OPTA Analyst 참고)
- 한국어, 캐주얼 X, 매거진 어조 (존댓말 X, 평어 O)
- 데이터가 주인공, 수식어는 보조
- "픽" / "강추" / "필승" 같은 마케팅 단어는 즉시 거부됨

# 재호출 정책 (Q-A 통합 카운트)
